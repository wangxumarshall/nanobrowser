# 「活着么」 (Still Alive)

> 一个“数字化的灵魂天平”，将对死亡的恐惧转化为对当下存续的积极确认。

## 项目简介
本项目是「死了么」APP 的反向概念实现，旨在通过生命倒计时和 AI 行为分析，帮助用户量化生命质量，寻找存在的意义。

## 技术栈
*   **前端框架**: React 18 + TypeScript
*   **构建工具**: Vite
*   **样式库**: Tailwind CSS
*   **动画库**: Framer Motion
*   **状态管理**: Zustand (持久化存储)
*   **国际化**: i18next (中/英)

## 目录结构
*   `web/`: 前端应用程序源代码
*   `SPECIFICATION.md`: 产品详细规格说明书

## 快速开始 (Quick Start)

### 1. 安装依赖
进入 web 目录并安装依赖包：

```bash
cd web
npm install
```

### 2. 启动开发服务器
启动本地开发环境，支持热更新：

```bash
npm run dev
```
启动后访问终端显示的地址（通常是 `http://localhost:5173`）。

### 3. 构建生产版本
构建用于部署的静态文件：

```bash
npm run build
```
构建产物将输出到 `web/dist` 目录。

### 4. 预览构建结果
预览构建后的生产版本：

```bash
npm run preview
```

## 功能特性
1.  **生命倒计时**: 基于出生日期和预期寿命计算剩余秒数。
2.  **AI 灵魂日记**: 记录每日反思，模拟 AI 分析（本地逻辑）判断行为对生命的增减益。
3.  **意义清单**: 设定人生目标，达成后获得生命时间奖励。
4.  **沉浸式体验**: 深空黑主题与呼吸感交互动画。

## 部署
本项目构建后为纯静态资源，可部署至 Vercel, Netlify, GitHub Pages 或任何静态 Web 服务器。
